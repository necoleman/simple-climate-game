# Climate study


## Setting up pipenv

```bash
sudo apt-get install pipenv
PIPENV_VENV_IN_PROJECT=True pipenv install python 3.10
```

## Overview

Vizzies using pyglet
Research and modeling using jupyter

## Use

Exploring research notebooks:
```bash
pipenv run jupyter-lab
```
